# Firebase Studio

This is a NextJS e-commerce app deployed on Firebase.

## Live URL

https://safe-bookks-eu-72339747-5df54.web.app

## Features

- Next.js 14 with TypeScript
- Firebase Hosting, Firestore, Authentication
- Tailwind CSS
- React Hook Form with Zod validation

## Getting Started

To get started, take a look at src/app/page.tsx.

## Deployment

The app is deployed with Firebase Hosting. For full functionality including dynamic routes, ensure the Service Account User role is assigned to the deployer account, then run `firebase deploy` to deploy functions.
